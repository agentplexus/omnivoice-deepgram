## [v0.1.0] - 2026-01-19

### Highlights

- OmniVoice STT provider for Deepgram with real-time streaming transcription support

### Added

- Streaming STT provider implementing `stt.StreamingProvider` interface
- Real-time transcription via Deepgram WebSocket API
- Support for telephony audio formats: mu-law, a-law, linear16, FLAC, Opus, MP3
- Interim and final transcription results with word-level timing
- Speech start/end detection via `SpeechStarted` and `UtteranceEnd` callbacks
- Speaker diarization support with configurable max speakers
- Keyword boosting for domain-specific vocabulary
- Configuration converter from OmniVoice `TranscriptionConfig` to Deepgram options

### Documentation

- README with installation, usage examples, and configuration reference
- Package documentation with streaming transcription examples
