## [v0.1.0] - 2026-01-19

### Highlights

- OmniVoice STT provider for Deepgram with real-time streaming transcription support

### Added

- Streaming STT provider implementing `stt.StreamingProvider` interface ([`f8fb090`](https://github.com/agentplexus/omnivoice-deepgram/commit/f8fb090))
- Real-time transcription via Deepgram WebSocket API ([`f8fb090`](https://github.com/agentplexus/omnivoice-deepgram/commit/f8fb090))
- Support for telephony audio formats: mu-law, a-law, linear16, FLAC, Opus, MP3 ([`f8fb090`](https://github.com/agentplexus/omnivoice-deepgram/commit/f8fb090))
- Interim and final transcription results with word-level timing ([`f8fb090`](https://github.com/agentplexus/omnivoice-deepgram/commit/f8fb090))
- Speech start/end detection via `SpeechStarted` and `UtteranceEnd` callbacks ([`f8fb090`](https://github.com/agentplexus/omnivoice-deepgram/commit/f8fb090))
- Speaker diarization support with configurable max speakers ([`f8fb090`](https://github.com/agentplexus/omnivoice-deepgram/commit/f8fb090))
- Keyword boosting for domain-specific vocabulary ([`f8fb090`](https://github.com/agentplexus/omnivoice-deepgram/commit/f8fb090))
- Configuration converter from OmniVoice `TranscriptionConfig` to Deepgram options ([`f8fb090`](https://github.com/agentplexus/omnivoice-deepgram/commit/f8fb090))
